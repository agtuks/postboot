Introduction

Library that extends the Bootstrap framework.

Bootstrap version compatibility 4.0.0-alpha.6.
  • postboot/
    • css/
      • postboot.css
      • postboot.min.css
    • js/
      • postboot.js
      • postboot.min.js
<!DOCTYPE html>
<html lang="en">
  <head>
    <meta charset="utf-8">
    <meta name="viewport" content="width=device-width, initial-scale=1, shrink-to-fit=no">

	<!-- CSS -->
    <link rel="stylesheet" href="https://maxcdn.bootstrapcdn.com/bootstrap/4.0.0-alpha.6/css/bootstrap.min.css">
    <link rel="stylesheet" href="https://cdn.rawgit.com/alextarkhov/postboot/v1.0.0-alpha4/dist/css/postboot.min.css">
  </head>
  <body>
    <h1>PostBoot</h1>

	<!-- JS -->
    <script src="https://cdn.rawgit.com/alextarkhov/postboot/v1.0.0-alpha4/dist/js/postboot.min.js"></script>
  </body>
</html>
